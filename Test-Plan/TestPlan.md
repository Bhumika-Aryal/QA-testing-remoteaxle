# Test Plan – RemoteAxle Website

## Objective

The objective of this testing activity is to evaluate the functionality, usability, and reliability of the RemoteAxle platform and identify defects that could impact user experience.

## Scope

The following modules were included in testing:

- Homepage
- Footer links
- Course listing
- Course details page
- Send enquiry form
- Cart functionality
- Sign up module
- Transaction page
- Notifications
- Classroom section

## Testing Types Performed

Exploratory Testing  
Functional Testing  
UI Testing  
Validation Testing  
Basic Performance Observation

## Test Environment

Browser: Google Chrome  
Operating System: Windows 11  
Network: Standard broadband connection

## Entry Criteria

- Website should be accessible
- Homepage should load successfully

## Exit Criteria

- All planned modules tested
- Identified defects documented
- Bug report prepared
